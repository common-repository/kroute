== Installation ==

= Minimum Requirements =

* WordPress 4.4 or greater
* PHP version 5.2.4 or greater

== Description ==

A plugin that supply shortcodes to get current dynamic URLs on wordpress editor.

= Docs & Support =

- Use shortcode [k_route] to get main site URL.
- Use pageid parameter to get concrete url by id, [k_route pageid=6]
- Use pageslug parameter to get an url by slug, [k_route pageslug=example]
